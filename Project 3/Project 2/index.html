日本地理位置

日本位於北緯20°至60°、東經123°到154°之間，為亞洲東面的島國。 東濱太平洋，與美國、加拿大相遙望；西臨日本海、黃海與朝鮮半島、中國大陸為鄰；南隔太平洋與台灣咫尺相鄰；北接鄂霍次克海和俄羅斯互峙。 由北海道、本州、四國、九州四個大島及延長約三千公里的四千多個島嶼所組成，故有「島國」之稱。
